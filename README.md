# GenScriptAI

# 1. AI Song Generation
This is an end-to-end multi-modal AI system that converts user-provided text prompts into complete songs. It leverages large language models to generate structured lyrics (such as verses and choruses) based on a given theme, and then uses text-to-speech (TTS) techniques to synthesize these lyrics into audio. The system integrates multiple AI components into a unified pipeline, handling input processing, lyric generation, audio synthesis, and final output creation.

The project focuses on AI pipeline orchestration and multi-modal integration, demonstrating how NLP and audio processing can be combined to build a functional, real-world content generation tool.

**Tech:** Python, LLM APIs, Text-to-Speech (TTS), Pydub, Jupyter/Colab, Prompt Engineering, API Integration

**Link:** https://github.com/Shubham-kumar6398/Ai_song_generator

# 2. AI Video Generation
This is a multi-modal AI system that transforms text prompts or stories into complete videos. It uses large language models to break down the input into structured scene descriptions, which are then converted into images using diffusion-based image generation models. These images are sequentially combined to form a video, with added narration or background audio generated through text-to-speech techniques.

The project demonstrates end-to-end integration of NLP, computer vision, and audio processing, showcasing how multiple AI components can be orchestrated to automate video content creation.

**Tech:** Python, LLM APIs, Stable Diffusion (Diffusers), MoviePy/OpenCV, TTS, Pydub, NumPy, Jupyter/Colab, Prompt Engineering, API Integration

**Link:** https://github.com/chaitanyakumarcodes/ai_video_generator
